## 📝 CONTENT MANAGEMENT

### Adding Experience
Edit `data/experience.json`:
```json
{
  "company": "Company",
  "title": "Job Title",
  "duration": "Start - End",
  "description": "What you did",
  "highlights": ["achievement 1", "achievement 2"]
}
```

### Adding Projects
Edit `data/projects.json`:
```json
{
  "name": "Project Name",
  "description": "What it does",
  "technologies": ["Tech1", "Tech2"],
  "impact": "Optional: The results"
}
```

### Adding Articles
Edit `data/articles.json`:
```json
{
  "date": "2024-01-15",
  "title": "Article Title",
  "excerpt": "Summary",
  "link": "https://your-article",
  "tags": ["tag1", "tag2"]
}
```

**See CONTENT_GUIDE.md for complete examples!**

## 🎨 CUSTOMIZATION
